# Dashboard de Assinaturas no Excel

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um dashboard interativo no Microsoft Excel para análise de informações relacionadas a assinaturas de um serviço. O objetivo foi transformar dados brutos em indicadores visuais que facilitam o acompanhamento das principais métricas e auxiliam na tomada de decisão.

## 🎯 Objetivos

- Organizar e analisar dados utilizando o Microsoft Excel;
- Construir um dashboard com indicadores e gráficos dinâmicos;
- Facilitar a interpretação dos dados por meio de visualizações;
- Consolidar conhecimentos em análise de dados com Excel.

## 🛠️ Ferramentas utilizadas

- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Segmentação de Dados (Slicers)
- Formatação Condicional
- Fórmulas e Funções

## 📊 Funcionalidades

- Dashboard interativo;
- Indicadores (KPIs);
- Filtros por segmentação de dados;
- Gráficos dinâmicos;
- Organização das informações em painel de controle.

## 💡 Principais aprendizados

Durante o desenvolvimento deste projeto foi possível praticar:

- Organização e tratamento de dados;
- Construção de dashboards no Excel;
- Utilização de Tabelas Dinâmicas;
- Criação de gráficos para análise de indicadores;
- Boas práticas de apresentação de dados.

## 📈 Indicadores analisados

O dashboard permite acompanhar métricas como:

- Receita total;
- Quantidade de assinantes;
- Distribuição por tipo de plano;
- Evolução dos resultados;
- Comparação entre categorias.
